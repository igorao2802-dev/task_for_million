Чтобы переопределить встроенный стиль `style="color: green !important;"` без удаления его и без использования JavaScript, можно использовать **более специфичный CSS-селектор** и также применить `!important`. В CSS правило с большей специфичностью и равным приоритетом `!important` победит.

Вот пример такого CSS:

```css
html body p {
  color: red !important;
}
```

### Объяснение:
- Селектор `html body p` имеет **большую специфичность**, чем просто `p`.
- Использование `!important` на этом правиле позволяет конкурировать с встроенным стилем.
- В большинстве браузеров это правило **перезапишет встроенный стиль**, если специфичность окажется выше.

Если это не сработает (в зависимости от браузера), можно усилить специфичность ещё больше:

```css
body p.my-paragraph {
  color: blue !important;
}
```

И в HTML:

```html
<p class="my-paragraph" style="color: green !important;">Текст</p>
```

Или даже так:

```css
body body body p {
  color: orange !important;
}
```

Чем больше конкретики — тем выше шанс переопределения. Хочешь, покажу, как это выглядит в реальном примере?
