Задача состоит в том, чтобы "перебить" встроенный стиль через другие способы задания стилей. JavaScript использовать нельзя. Изменять html-код оиписания элемента нельзя. Остальные абзацы не должны менять цвет! Исходный html-код со встроенным элементом:
```
<p style="color: green !important;">Цвет этого текста нужно изменить</p>
```
